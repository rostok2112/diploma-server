# diploma-server
Server for reading, modifying and transfering data from FBT module to SteamVR

---

Server side of [FBT module project](https://github.com/rostok2112/diploma)

Dependencies:
- pipenv


Initialization:

```bash
pipenv sync
```

Run:

```bash
pipenv run python src/main.py
```

Exit:

`CTRL + C`
